# Personal Data Management System

A full stack web application that allows users to store, manage, update, and view personal and educational information with image upload support.

## Features

- Add personal details
- Store educational information
- Upload profile images
- Edit and update records
- View saved user data
- Responsive design

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Project Structure

- Backend developed using Django
- Frontend built with HTML, CSS, and JavaScript
- SQLite used for database management

## How to Run

```bash
pip install -r requirements.txt
python manage.py runserver
```
